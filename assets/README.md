# Assets Folder

This folder contains visual outputs and screenshots from the NeuroNova system.

## Required Images

Please add the following images to this folder:

1. **screenshot1.png** - First screenshot of the conversation graph visualization
   - Should show the emotion activation heatmap, mental health risk, firing rates, and membrane potentials
   - Recommended size: 500px width for README display

2. **screenshot2.png** - Second screenshot of the conversation graph visualization
   - Should show a different state or conversation turn
   - Recommended size: 500px width for README display

## How to Add Images

Simply drag and drop your screenshot images into this folder and name them:
- `screenshot1.png`
- `screenshot2.png`

The images will automatically display in the main README.md file.

## Generated Files

When you run `examples/demo_noninteractive.py`, the following files will be generated here:
- `transcript.txt` - Conversation transcript
- `volt_example_1.png` - Membrane potential plot for example 1
- `volt_example_2.png` - Membrane potential plot for example 2
- `volt_example_3.png` - Membrane potential plot for example 3
- `conversation_graph.png` - Full conversation graph visualization
